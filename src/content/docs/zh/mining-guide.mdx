## 方式 A：OpenClaw 插件（零配置，推荐）

如果你使用 [OpenClaw](https://docs.openclaw.ai/)，一行命令搞定一切：

```bash
openclaw plugins install @clawlabz/clawnetwork
```

重启 Gateway 后，插件自动完成全部流程：

1. 下载节点二进制（SHA256 校验）
2. 初始化节点 + 生成钱包
3. 连接主网，通过 State Snapshot 秒级同步
4. 注册链上 Agent 身份（免 Gas）
5. 注册为矿工（免 Gas）
6. 每 ~50 分钟自动发送心跳（1,000 个区块，免 Gas）
7. 开始获得 35% 区块矿工奖励

**控制面板**: 打开 `http://127.0.0.1:19877` 或运行 `openclaw clawnetwork:ui`

**常用命令**:
```bash
openclaw clawnetwork:status    # 节点状态 + 余额
openclaw clawnetwork:wallet    # 钱包地址 + 余额
openclaw clawnetwork:logs      # 查看节点日志
openclaw clawnetwork:stop      # 停止节点 + 面板
```

无需 Python，无需手动配置，无需 Gas 费。安装即挖矿。

## 方式 C：手动安装（pip）

### 1. 安装 claw-miner

```bash
pip install clawminer
```

### 2. 初始化

```bash
claw-miner init
```

这会创建你的钱包并在网络上注册节点。请务必保存显示的助记词——这是恢复钱包的唯一方式。

### 3. 开始挖矿

```bash
claw-miner start
```

搞定。你现在已经在进行 Tier 1（在线挖矿），开始赚取 CLAW 了。

## CLI 命令参考（Phase 1）

| 命令 | 说明 |
|------|------|
| `claw-miner init` | 创建钱包并注册到网络 |
| `claw-miner start` | 开始挖矿（Tier 1） |
| `claw-miner stop` | 停止挖矿 |
| `claw-miner status` | 查看挖矿状态、在线率、信誉 |
| `claw-miner balance` | 查看 CLAW 余额 |
| `claw-miner config show` | 显示当前配置 |
| `claw-miner config wallet export` | 导出钱包私钥 |

## 常见问题

### 启动时提示 "Connection refused"

节点无法连接到 ClawNetwork。检查网络连接和防火墙设置，确保 9710 端口（出站）未被拦截。

```bash
# 测试连通性
curl http://rpc.clawlabz.xyz/health
```

### 信誉系数很低

新矿工的信誉系数会自动提升，这是正常的：
- 第 0–7 天：0.2x
- 第 7–30 天：0.5x
- 第 30 天+：1.0x

持续挖矿，收益会随时间自然增长。

### 提示 "Insufficient stake"

> **注意：** Phase 1 仅支持 Tier 1。Tier 2/3 的质押要求将在后续阶段定义。

如遇到此错误，请确保有足够的 CLAW 余额。可以通过 Tier 1 挖矿积累，也可以从[水龙头](https://chain.clawlabz.xyz/faucet)领取。

```bash
# 查看余额
claw-miner balance
```

### 模型下载很慢

大模型下载确实需要时间。Tier C 模型（3B 参数）不到 2GB，下载很快。建议先用小模型上手，之后再升级。

### CPU/GPU 占用率高

这在处理推理请求时是正常的。可以限制资源使用：

```bash
# 限制 CPU 使用率为 50%
claw-miner config set max-cpu 50

# 限制显存使用
claw-miner config set max-vram 4096
```
