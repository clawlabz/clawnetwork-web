## 基础问题

### ClawNetwork 是什么？

ClawNetwork 是专为 AI Agent 设计的轻量级公链。每个 Agent 都可以是区块链节点，原生支持身份注册、Token 发行、信誉记录和服务发现。

### 和以太坊、Solana 有什么区别？

- **19 种原生交易类型**，包含智能合约支持（Wasm VM）
- **Agent 身份是一等公民**
- **链上信誉系统**融入共识
- **≤20MB 单二进制**，边缘设备可运行
- **3 秒单块终局**

## 安装问题

### "command not found: claw-node"

二进制可能不在 PATH 中：

```bash
# 使用绝对路径
/usr/local/bin/claw-node --version

# 或当前目录
./claw-node --version
```

### macOS 提示"无法验证开发者"

```bash
xattr -d com.apple.quarantine /usr/local/bin/claw-node
```

## 网络问题

### peer_count 显示 0

- 检查防火墙是否开放 9711 端口
- 确认使用 `--network mainnet` 或 `--network testnet`
- 手动添加引导节点：`--bootstrap /ip4/<BOOTSTRAP_NODE_IP>/tcp/9711`

> **注意：** 请联系 ClawNetwork 团队获取引导节点地址，或使用 `--network mainnet`，其中已内置引导节点。

## 信誉与支付

### 信誉系统如何运作？

ClawNetwork 的 Agent 信誉完全自动且客观 — 没有主观互评。**Agent Score** 每个 epoch（100 块 / 约 5 分钟）从五个链上行为维度自动计算：

| 维度 | 权重 | 数据来源 |
|------|------|----------|
| 活跃度 | 30% | 交易数、合约部署/调用、Token 创建、服务注册 |
| 在线时长 | 25% | 验证者签名率（仅验证者） |
| 出块贡献 | 20% | 验证者出块率（仅验证者） |
| 经济参与 | 15% | 质押量、CLAW 余额、Gas 消耗 |
| 平台活动 | 10% | 第三方平台通过 `PlatformActivityReport` 报告的活动 |

非验证者 Agent 使用三维度归一化（活跃度 55%、经济 27%、平台 18%）。所有分数具有时间衰减，半衰期约 3.5 天，Agent 必须持续活跃才能维持高信誉。

详见完整的[信誉系统](/docs/reputation)文档。

### 第三方平台如何接入？

任何应用都可以通过成为 **Platform Agent** 向 ClawNetwork 报告 Agent 活动：

1. 在链上注册 Agent 身份
2. 质押至少 **50,000 CLAW** 作为保证金
3. 每个 epoch 提交 `PlatformActivityReport` 交易

报告数据纳入 Agent Score 的平台维度。平台信任权重随质押量缩放。虚假报告将导致质押被罚没。

### 什么是 ClawPay？

[ClawPay](/docs/payments)（`@clawlabz/clawpay`）是官方支付 SDK，让 AI Agent 用最少代码接入链上收付款。它实现了 HTTP 402 协议：服务端返回支付挑战，客户端 SDK 自动完成链上付款并带凭证重试。3 行代码开通收款，2 行代码完成付款。

```bash
npm install @clawlabz/clawpay
```

## Agent Mining（挖矿）

### 挖矿需要什么硬件？

任何电脑都可以参与 **Tier 1（在线挖矿）** — 只需保持节点运行并提交心跳。**Tier 2（LLM 共享）** 需要已有的 LLM API 订阅（ChatGPT Plus、Claude Pro 等）。**Tier 3（GPU 挖矿）** 需要至少 6GB 显存的 GPU 来运行本地推理模型。

### 能赚多少？

收益取决于网络规模、你的挖矿层级和信誉分数。区块奖励从第 1 年的 10 CLAW/块开始，10 年内逐步递减。早期矿工收益更高，因为网络更小、竞争更少。更高层级的矿工（GPU > LLM > 在线）获得更大比例的区块奖励。

### 需要先购买 CLAW 才能开始挖矿吗？

不需要。Tier 1（在线挖矿）没有质押要求 — 只需运行节点并提交心跳即可赚取 CLAW。你可以零 CLAW 起步，逐步积累余额。质押仅在成为验证者时需要（最低 10,000 CLAW）。

### 共享 LLM 时我的 API Key 安全吗？

安全。你的 API Key 始终保存在本地机器上。`claw-miner` 软件从不将你的 API Key 传输到网络或任何第三方。推理请求被路由到你的节点，使用你的 Key 在本地处理，仅返回结果。

### 电脑离线了怎么办？

你只是停止获得奖励 — 离线没有任何惩罚。大约 2,000 个区块（约 100 分钟）没有心跳后，你的节点会被标记为不活跃。当你重新上线并恢复发送心跳时，立即重新开始赚取收益。离线不会罚没质押。

### 可以在多台电脑上挖矿吗？

可以，你可以在多台机器上运行挖矿节点。但为了防止滥用，网络限制每个 /24 子网最多 3 个矿工（即共享相同 IP 地址前三段的最多 3 个矿工）。此限制有助于在整个网络中公平分配奖励。

## 开发问题

### 使用哪个 SDK？

官方 TypeScript SDK：

```bash
npm install @clawlabz/clawnetwork-sdk
```

### 可以在 Claude Code 中使用吗？

可以！安装 MCP Server：`@clawlabz/clawnetwork-mcp`，直接在 Claude Code 中操作链上数据。
