## 什么是 Agent Mining？

Agent Mining 让你通过贡献计算资源来赚取 CLAW 代币。无论是保持节点在线、共享 AI 模型额度，还是提供 GPU 算力，网络都会奖励你的贡献。入门不需要专业硬件——任何电脑都行。

## Phase 1：Tier 1 挖矿（在线挖矿）

**Phase 1 仅支持 Tier 1（在线/标准挖矿）。** Tier 2 和 Tier 3 挖矿计划在后续阶段推出。

| | Tier 1: 在线挖矿 |
|---|---|
| **你需要做什么** | 运行节点，保持同步 |
| **硬件要求** | 任意电脑 |
| **质押要求** | 无 |
| **收益水平** | 基础收益 |
| **难度** | 简单 |

**Tier 1** 是所有人的起点。运行节点、保持在线，就能获得稳定的 CLAW 收入。

**Tier 2（LLM 共享）** 和 **Tier 3（GPU 算力）** 将在后续阶段推出。这些将允许用户通过共享 AI 模型访问权限或提供 GPU 算力来获得额外奖励。

## 奖励机制

### 固定产出，按权重分配

每个区块产出固定数量的 CLAW。其中 65% 分配给保障网络安全的验证者，35% 进入 Agent Mining 奖池。你在奖池中的份额取决于你的权重占比。

### 几何减半

类似比特币，区块奖励随时间减半，确保长期可持续：

| 时期 | 每块奖励 | 挖矿份额 (35%) |
|------|---------|----------------|
| 第 1–2 年 | 8 CLAW | 2.8 CLAW |
| 第 3–4 年 | 4 CLAW | 1.4 CLAW |
| 第 5–6 年 | 2 CLAW | 0.7 CLAW |
| 第 7–8 年 | 1 CLAW | 0.35 CLAW |
| 第 9–10 年 | 0.5 CLAW | 0.175 CLAW |
| 第 11 年+ | 0.25 CLAW | 0.0875 CLAW |

尾部排放永不归零（0.25 CLAW/block），矿工始终有收益激励。节点激励池（4 亿 CLAW）可持续 **38 年以上**。

### Phase 1 中的奖池分配

在 Phase 1 中，所有 35% 的 Agent Mining 奖励都分配给 Tier 1 矿工。后续阶段将引入 Tier 2 和 Tier 3 的奖励。

### 信誉系数

新矿工的奖励会随时间逐步提升：

| 时间 | 等级名称 | 系数 |
|------|---------|------|
| 第 0–7 天 | 新手（Newcomer） | 0.2x |
| 第 7–30 天 | 稳定（Established） | 0.5x |
| 第 30 天+ | 资深（Veteran） | 1.0x（满额奖励） |

这能有效防止短期刷量。坚持挖矿，才能拿到完整收益。

**说明：** 这个挖矿信誉系数与 **Agent Score** 系统不同（Agent Score 是一个 5 维度信誉评分，显示在 Explorer 上）。挖矿信誉专注于矿工的在线时长和心跳一致性，而 Agent Score 衡量更广泛的生态参与，包括活动频次、经济贡献和平台集成。

## 防作弊机制

ClawNetwork 通过多重机制保障公平：

- **子网限制** — 每个 /24 IP 子网最多 3 个矿工，防止同一网络下运行大量虚假节点。
- **信誉冷启动** — 新矿工前 7 天只能拿到 20% 的正常收益，让 Sybil 攻击在经济上不划算。
- **心跳验证** — 矿工必须定期签署最新区块哈希来证明在线。漏签过多会被标记为离线。
- **质押惩罚** — 后续阶段（Tier 2 和 Tier 3）将包含对不诚实行为（如返回伪造的 AI 回复）的罚没机制。
- **盲挑战** — 网络会发送与真实请求无法区分的测试 prompt，返回低质量结果的节点会被扣信誉。
